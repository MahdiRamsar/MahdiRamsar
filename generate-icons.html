🎨 ساخت آیکون‌های PWA

برای سایت شخصی مهدی رحیمیان — با یک کلیک دانلود کنید

📁 نحوه استفاده

  • روی دکمه دانلود زیر هر آیکون کلیک کنید
  • فایل‌های icon-192.png و icon-512.png را در پوشهٔ پروژه قرار دهید
  • فایل manifest.json را در همان پوشه ایجاد کنید (متن آن در پایین آمده)
  • سایت را با HTTPS اجرا کنید تا PWA فعال شود

📄 محتوای manifest.json

{ "name": "مهدی رحیمیان | سایت شخصی", "short_name": "مهدی رحیمیان", "description": "سایت شخصی مهدی رحیمیان", "start_url": "/", "display": "standalone", "background_color": "#07080b", "theme_color": "#07080b", "icons": [ { "src": "icon-192.png", "sizes": "192x192", "type": "image/png" }, { "src": "icon-512.png", "sizes": "512x512", "type": "image/png" } ] }

📄 service worker (sw.js)

const CACHE_NAME = 'mahdi-site-v1'; const urlsToCache = ['/', '/index.html', '/icon-192.png', '/icon-512.png']; self.addEventListener('install', e => { e.waitUntil(caches.open(CACHE_NAME).then(c => c.addAll(urlsToCache))); }); self.addEventListener('fetch', e => { e.respondWith(caches.match(e.request).then(r => r || fetch(e.request))); });
